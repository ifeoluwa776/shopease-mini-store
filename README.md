# ShopEase Mini E-Commerce Store

ShopEase is a mini e-commerce store built with Next.js and React. It allows users to browse products, view product details, add items to a cart, and manage their cart with persistent storage.

## Features

- View featured products from the homepage
- View product details with dynamic routing
- Add items to cart
- Remove items from cart
- View total cart price
- Cart data persists with localStorage
- Global cart state managed with Context API

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
